# Images Directory

Place your three emotional state images here:

- `neutral.png` or `neutral.jpg` - Neutral state (0)
- `positive.png` or `positive.jpg` - Positive/Manic state (3)
- `depressive.png` or `depressive.jpg` - Very Depressive state (-3)

The website will automatically display the appropriate image based on the current state.

Supported formats: PNG, JPG, JPEG, WebP
